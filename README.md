# 1. 프로젝트 선정 이유
#### 
대학 전공 선택은 개인의 진로와 직결되는 매우 중요한 결정입니다.
그러나 많은 학생들이 자신의 성향을 충분히 분석하지 못한 채 성적이나 주변의 권유에 따라 전공을 선택하는 경우가 많습니다.

학생들의 진로 선택에 조금이나마 도움이 되고자, 본 프로젝트는 사용자의 성향을 수치화하여 데이터 기반으로 분석하고 이에 적합한 대학 전공을 추천하는 프로그램을 구현하는 것을 목표로 선정하였습니다.

본 프로젝트는 다음과 같은 특징을 가집니다.

- 객관적인 수치 기반 전공 추천
- 사용자 만족도 피드백을 반영한 반복 추천 구조
- 주전공 확정 이후 복수전공까지 고려한 확장 설계

이를 통해 실제 전공 탐색 과정과 유사한 추천 시스템을 구현하고자 하였습니다.


# 2. 코드 설명

### 1. 전공 데이터 구성
본 프로그램에서는 총 30개의 대학 전공 데이터를 사전에 정의하였습니다.
각 전공은 다음 9가지 기준을 바탕으로 1~10 스케일의 점수로 수치화되어 있습니다.

1. 팀플 선호
2. 프로젝트 선호
3. 학습량
4. 취업 선호
5. 발표 선호
6. 창의성
7. 실습 선호
8. 진로 다양성
9. 사고력

이를 통해 각 전공의 특성을 정량적으로 표현하였으며,
해당 데이터는 이후 KNN 기반 전공 추천 알고리즘의 입력값으로 활용됩니다.

### 2. 사용자 성향 입력
사용자는 자신의 학습 및 활동 성향을 각 기준별로 1~10 스케일로 입력하게 됩니다.
입력값이 범위를 벗어날 경우 재입력을 요구하도록 하여, 잘못된 입력으로 인한 프로그램 오류를 방지하였습니다.
입력된 사용자 성향 데이터는 이후 모든 추천 과정에서의 정보로 사용됩니다.


### 3. KNN 모델 초기화 및 학습
전공 추천을 위해 KNN(K-Nearest Neighbors) 알고리즘을 사용하였습니다.
StandardScaler를 활용하여 전공 데이터를 정규화하였습니다.
유클리디안 거리(Euclidean Distance)를 기준으로 사용자 성향과 전공 간의 유사도를 계산하고,
전체 전공 데이터를 기반으로 KNN 모델을 학습시켰습니다.
이를 통해 사용자 성향과 가장 유사한 전공을 효과적으로 탐색할 수 있도록 구현하였습니다.


### 4. 전공 추천 함수
사용자의 성향 벡터를 학습된 KNN 모델에 적용하여 적합도가 높은 전공 TOP 5를 도출합니다.
- 이미 추천된 전공은 제외 목록(set)으로 관리됩니다.
- 거리 값을 기반으로 적합도를 계산하여 백분율 형태로 변환합니다.
- 학과명, 전공 분류, 적합도를 표 형태로 출력합니다.


### 5. 만족도 입력 및 피드백 수집
추천된 전공에 대해 사용자는 만족도(1~10)를 입력하게 됩니다.
- 만족도가 10점인 전공이 존재할 경우 해당 전공을 주전공으로 확정합니다. 
- 만족도 10점인 전공이 존재하지 않을 경우 성향 업데이트를 하여 다음 추천을 진행합니다.


### 6. 성향 업데이트 함수
최초 추천 과정에서 주전공이 확정되지 않을 수 있으므로,
입력된 만족도를 기존 사용자 성향에 반영하여 성향을 조정합니다.

조정된 성향을 기반으로 다시 전공 추천을 수행함으로써,
사용자의 취향을 점진적으로 반영하는 추천 구조를 구현하였습니다.

### 7. 메인 함수에 대한 설명
메인 함수에서는 프로그램의 전체 흐름을 제어합니다.
1. 사용자 성향 입력
2. 제외할 전공 목록(set) 초기화
3. 주전공이 확정될 때까지 while문을 통한 반복 추천
4. 추천 결과를 표 형태로 출력
5. 만족도 입력 및 성향 업데이트 수행
6. 주전공 확정 시 복수전공 추천 여부 확인
7. 복수전공 추천을 선택한 경우에는 추천 후 프로그램 종료, 복수전공 추천을 선택하지 않은 경우에는 그대로 프로그램 종료

# 4. 프로젝트 결론 및 총평
#### 
본 프로젝트는 대학 전공 선택이라는 현실적인 고민을 데이터 기반으로 접근하여 해결해 보고자 한 점에서 의의를 가집니다.
대학 전공은 개인의 학업 방향과 진로에 큰 영향을 미치지만, 많은 학생들이 자신의 성향을 충분히 분석하지 못한 채 전공을 선택하는 경우가 많습니다.
이에 본 프로젝트는 사용자의 성향을 수치화하여 객관적으로 분석하고, 이에 적합한 대학 전공을 추천함으로써 전공 탐색 과정에 하나의 참고 기준을 제시하고자 하였습니다. 
또한 추천 결과에 대한 만족도 피드백을 반영하여 성향을 조정하고 재추천을 함으로써, 전공 선택의 폭을 넓히고자 했습니다.
주전공이 확정된 이후에는 복수전공 추천 기능을 제공하여, 실제 대학 생활에서 고려되는 전공 간의 조합과 진로 다양성까지 함께 고려할 수 있도록 했습니다.
기술적인 측면에서는 KNN 알고리즘을 활용한 거리 기반 유사도 분석을 적용하여 추천 결과의 객관성과 확장성을 확보하였습니다.
본 프로젝트는 프로그래밍 기술을 활용하여 학생들이 겪는 전공 선택 문제를 데이터와 알고리즘 관점에서 해결해 보고자 한 의미 있는 경험이었다고 생각합니다.
